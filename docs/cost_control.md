\# Cost Control Plan



This project is designed for learning using free or trial-based Microsoft services.



\## Free / Trial Tools Used



\- Microsoft Fabric Trial

\- Power BI Desktop

\- GitHub Free

\- Small sample CSV data



\## Cost Control Rules



1\. Use only small sample CSV files.

2\. Do not use real banking data.

3\. Do not schedule pipelines frequently.

4\. Run Fabric notebooks manually during development.

5\. Avoid paid Fabric capacity.

6\. Avoid Azure Databricks, Synapse dedicated pools, and virtual machines.

7\. Export project artifacts and screenshots before trial expiry.

8\. Delete unused Fabric items after project completion.

9\. Store only code, documentation, and sample data in GitHub.



\## Not Used



\- Real customer data

\- Paid Power BI Pro sharing

\- Paid Fabric capacity

\- Azure Databricks

\- Azure Synapse dedicated SQL pool

\- Azure virtual machines
